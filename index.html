Style Guide & Components

To be used for Live Suite Software

The intention of this guide is both to ensure that user experience is consistent throughout all of Live Suite software and to provide a visual reference for UI dev

Colors

Using Colors

To set background color

Use class name

<div class="dm-blue"></div>

To set text color

Use class name and add "-text"

<div class="dm-blue-text">/div>

Default Black & Grays

"dm black" is our black and should always be set as the default color in the body tag. The varying shades of grays can be used as an alternative to dm-black or white.

dm-black #262c37

dm-gray-darker   #7B8289

dm-gray-dark   #A9B0b7

dm-gray   #DCE0e2

dm-gray-light   #EBEff2

dm-white  #ffffff

Shadows & Outline

Use these classes to simply add a pre-designed border or a shadow around a box

Adding an Edge

Basic Shadow

This shadow adds depth to any box and can be used for any purpose

example
<div class="shadow"></div>
{ box-shadow: $shadow-normal }
Deep Shadow

This shadow adds more depth and is best used for larger boxes

example
<div class="shadow-deep"></div>
{ box-shadow: $shadow-deep }
Tiny Shadow

This shadow a tiny bit of depth and can be a good alternative to an outline

example
<div class="shadow-small"></div>
{ box-shadow: $shadow-shallow }
Inset

This adds a glow to the inside for a pressed affect for pressed states

example
<div class="shadow-pressed"></div>
{ box-shadow: $shadow-inset }
Outline

As an alternative to a shadow, a box can be made clearer with simple outline

example
<div class="outline"></div>
{ border: $border-outline }

Grids & Tiles

Using one of the following grids will ensure all elements within the grids are evenly spaced and responsive to varying windows. The numbers correspond to the the maximum number of tiles when window is stretched out. In all cases, the number of columns will reduce to 2 when at 900 or less, and will span the full width at 600 or less

In css, when using display:grid it becomes possible to set row-gapandcolumn-gapin this case, both are set to$gap:2rem This ensures that both horizontal and vertical spacing stays consistent.

important When designing a tile or a card that will populate a grid, avoid setting a width or adding margin. Anything outside of the tile will be managed by the grid. This ensures the grid will display all its children correctly.

Buttons

Main Buttons

Main Buttons have a width set to 100% to ensure correct alignment when placed inside a column, which works nicely on mobile screens. A max width of 400px prevents button from getting ridiculously wide. Crucial to usability are feedback states (hover, pressed, etc) and are included in the class.

btn-action

Is blue and indicates progression or "next".

Submit 
<a class="btn-action">Submit</a>

btn-alert

Is red which and indicates caution before progressing.

Delete 
<a class="btn-alert">Abort</a>

btn-secondary

Is outlined which indicates an alternative to the main action, such as "cancel"

Cancel 
<a class="btn-secondary">Cancel</a>

Icons

DM icons are set in a font format called "dm-icons". Each icon is designed to be consistent in border width and visually clear no matter the size. Please use only DM designed icons in the UI.

View Complete Icon Set

Using icons

To add an icon, simply use the <i> tag with the icon name.

scale 
<i>scale</i>

Adding Color

Add a class to color the icon. Remember to color it like text.

scale 
<i class="dm-brand-red-text">scale</i>

Use Icon in CSS

When using in CSS, rather than treating it like an image and adding it to the background, include an icon by using the Unicode character as shown below. 

.example {
  background-color: $dm-gray-base;
  color: white;
  &::before {
    font-family: 'dm-icons';
    font-size: medium; 
    content: "\e9b1";
  }
}

Tile Indicators

important Elements using these indicators needs to have position set to 'relative'

Indicators with icons

Blue = There are new updates.

Red = Something is wrong.

Red indicator conveys 'errors' or 'safety problem' therefore it includes the 'bang' icon

Yellow = Heads up.

Yellow indicator conveys 'warning' or 'important' therefore it includes the 'bang' icon
